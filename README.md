# Pyjama DZ Pointeuse 🇩🇿 ⏰

Système moderne de gestion de pointage et de présence (Pointeuse) adapté pour les entreprises en Algérie (DZ).

## 🚀 Fonctionnalités Prévues
- **Pointage en temps réel** : Enregistrement des arrivées et départs.
- **Gestion des employés & plannings** : Suivi des horaires et rotations.
- **Rapports & Statistiques** : Calcul des heures travaillées, retards et absences.
- **Backend Supabase** : Base de données PostgreSQL en temps réel et authentification sécurisée.

## 🛠️ Technologies
- Frontend : Web App moderne (Vite / React / Tailwind CSS)
- Backend : Supabase (Auth, Database, RLS)
